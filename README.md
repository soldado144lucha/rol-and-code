# ROL & CODE

Intento de videjuego de mesa pero virtualizado.
Basado en el D&D,pero,con un tono directo a las FPs que hay en ESPAÑA.
El proyecto se ira agrandando hasta tener un buen videjuego con mas clases (más FPs) y diferentes mundo.
Desarrollado por un grupo de alumnos del IES Augustobriga.

# CUALQUIER PARECIDO CON LA REALIDAD ES PLENAMENTE PARA JUGAR.

An attempt at a virtualized tabletop board game.
Based on D&D, but with a direct spin on the Vocational Training degrees (FPs) in Spain. The project will gradually expand into a full-fledged video game, featuring more classes (additional FPs) and different worlds.
Developed by a group of students from IES Augustóbriga.

# ANY RESEMBLANCE TO REALITY IS PURELY FOR PLAYING PURPOSES.
